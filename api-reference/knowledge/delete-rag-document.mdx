---
title: "Delete RAG Document"
api: "DELETE /api/knowledge/documents/{id}"
description: "Deletes a document and removes all its vectors from the vector store"
---

## Path Parameters

<ParamField path="id" type="string" required>
  Document ID to delete
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the deletion was successful
</ResponseField>

<ResponseField name="message" type="string">
  Status message or error details
</ResponseField>

<RequestExample>

```bash cURL
curl -X DELETE https://api.example.com/api/knowledge/documents/doc_abc123 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch(
  'https://api.example.com/api/knowledge/documents/doc_abc123',
  {
    method: 'DELETE',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  }
);

const data = await response.json();
```

```python Python
import requests

response = requests.delete(
    'https://api.example.com/api/knowledge/documents/doc_abc123',
    headers={'Authorization': 'Bearer YOUR_API_KEY'}
)

data = response.json()
```

</RequestExample>

<ResponseExample>

```json Success Response
{
  "success": true,
  "message": "Document and associated vectors deleted successfully"
}
```

```json Not Found
{
  "success": false,
  "message": "Document not found"
}
```

```json Error Response
{
  "success": false,
  "message": "Failed to delete vectors from vector store"
}
```

</ResponseExample>
