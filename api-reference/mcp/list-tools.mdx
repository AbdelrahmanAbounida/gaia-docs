---
title: "List MCP Tools"
api: "GET /api/mcp/tools"
description: "Retrieves all available tools from a connected MCP server"
---

## Query Parameters

<ParamField query="id" type="string" required>
  MCP server ID
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the request was successful
</ResponseField>

<ResponseField name="tools" type="array">
  Array of available tools
  <Expandable title="Tool Object">
    <ResponseField name="name" type="string">
      Tool name
    </ResponseField>
    <ResponseField name="description" type="string">
      Tool description
    </ResponseField>
    <ResponseField name="inputSchema" type="object">
      JSON Schema defining the tool's input parameters
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash cURL
curl -X GET "https://api.example.com/api/mcp/tools?id=mcp_abc123" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch(
  'https://api.example.com/api/mcp/tools?id=mcp_abc123',
  {
    method: 'GET',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  }
);

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.example.com/api/mcp/tools',
    headers={'Authorization': 'Bearer YOUR_API_KEY'},
    params={'id': 'mcp_abc123'}
)

data = response.json()
```

</RequestExample>
