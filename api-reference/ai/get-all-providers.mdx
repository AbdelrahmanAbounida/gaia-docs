---
title: "Get All Providers"
api: "GET /api/ai/providers"
description: "Retrieves all available AI providers with their capabilities (Language, Embeddings, Image) and models"
---

## Response

<ResponseField name="success" type="boolean">
  Indicates if the operation was successful
</ResponseField>

<ResponseField name="modelsProviders" type="array">
  Array of AI model providers
  
  <Expandable title="properties">
    <ResponseField name="name" type="string">
      Provider name
    </ResponseField>
    
    <ResponseField name="capabilities" type="array">
      Array of capabilities (Language, Embeddings, Image)
    </ResponseField>
    
    <ResponseField name="models" type="array">
      Array of available models from this provider
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="vectorstoresProviders" type="array">
  Array of vector store providers
</ResponseField>

<RequestExample>

```bash cURL
curl --request GET \
  --url https://api.yourapp.com/api/ai/providers
```

```javascript JavaScript
const response = await fetch('https://api.yourapp.com/api/ai/providers', {
  method: 'GET'
});

const data = await response.json();
```

```python Python
import requests

url = "https://api.yourapp.com/api/ai/providers"
response = requests.get(url)
data = response.json()
```

</RequestExample>
