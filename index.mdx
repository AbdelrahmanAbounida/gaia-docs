GAIA AI is an **open-source platform for building intelligent AI agents** that connect to your data, tools, and <Link target="_blank" href="https://modelcontextprotocol.io/docs/getting-started/intro">
   MCPs
</Link>

. Whether you're building customer support bots, research assistants, or enterprise knowledge systems, GAIA AI gives you full control.

- **Knowledge integration** via RAG (Retrieval-Augmented Generation)
- **Tool connections** to APIs, databases, and services
- **MCP integrations** for external system communication
- **OpenAI-compatible APIs** for seamless integration
- **Local & Cloud** AI Models Providers

## Core Features

### Knowledge (RAG)

Search and retrieve information from your documents using semantic search. Support for multiple vector stores and embedding providers.

[Explore Knowledge →](/knowledge/overview)

### Model Context Protocol (MCP)

Connect your agents to external systems, databases, and APIs through standardized MCP servers.

[Explore MCP →](/mcp/overview)

### Custom Tools

Design and execute custom tools in TypeScript to extend your agent's capabilities with real-world actions.

[Explore Tools →](/tools/overview)

### OpenAI-Compatible Chat

Access your configured agents through OpenAI-compatible endpoints. Works with existing OpenAI clients and SDKs.

[Explore API →](/api-reference/chat/completions)

## Quick Example

```typescript
// Connect to your GAIA AI agent using OpenAI SDK
import OpenAI from 'openai';

const client = new OpenAI({
  baseURL: 'http://localhost:3030/api/v1/ai/chat',
  apiKey: 'your-api-key'
});

const response = await client.chat.completions.create({
  model: 'gpt-4o',
  messages: [
    { role: 'user', content: 'Search our documentation for RAG setup' }
  ]
});
```

Your agent automatically searches your knowledge base, uses available tools, and returns contextual responses.
