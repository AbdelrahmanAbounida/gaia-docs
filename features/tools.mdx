## Custom Tools

Create tools in TypeScript:
```typescript
// tools/weather.ts
export default {
  name: "get_weather",
  description: "Get current weather for a location",
  parameters: {
    type: "object",
    properties: {
      location: { type: "string" }
    },
    required: ["location"]
  },
  execute: async ({ location }) => {
    const res = await fetch(`https://api.weather.com?q=${location}`);
    return await res.json();
  }
}
```
